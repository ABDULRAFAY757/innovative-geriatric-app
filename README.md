# Innovative Geriatrics Medical App

Full-stack prototype focused on improving medication adherence and family oversight for elderly patients.

## Tech Stack
- Backend: FastAPI (Python)
- Frontend: React.js
- Database: SQLite (for prototype)
- Authentication: JWT or session-based (prototype)

## Core Features
- Medication reminders
- Family dashboard
- Centralized patient records

## How to Run
1. Start backend: `uvicorn app.main:app --reload`
2. Start frontend: `npm start` inside `frontend` folder
